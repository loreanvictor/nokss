Enough CSS to get started


{% markdown %} nokss is a drop-in stylesheet that works out of the box: there is no need for using special classes or attributes. In fact, no need to touch your HTML code at all. Instead, it relies on semantics of HTML itself to figure out how to style various elements, so the more [semantic](https://web.dev/learn/html/semantic-html/) your HTML is, the more you get out of nokss.
