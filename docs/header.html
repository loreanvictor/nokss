Enough CSS to get started


{% markdown %} [nokss](.) is a drop-in stylesheet that works out of the box: there is no need for using special classes or attributes. In fact, no need to touch your HTML code at all. That said, you can get more control if you write more [semantic HTML](https://web.dev/learn/html/semantic-html/). ```css @import 'https://unpkg.com/nokss/dist/nokss.css'; ``` {% endmarkdown %}
