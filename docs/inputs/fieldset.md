## Field Sets & Grouping

Use field sets to group some form controls together:

```html
<fieldset>
  <legend>Login</legend>
  <input type="email" placeholder="Email address" />
  <input type="password" placeholder="Password" />
  <menu role="group" align="end">
    <button>Cancel</button>
    <button>Login</button>
  </menu>
</fieldset>
```
